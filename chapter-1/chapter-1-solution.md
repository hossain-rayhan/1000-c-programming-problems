## Probem 2
Print `Hello World` using a C program.

**Solution:**
```c
#include<stdio.h>
int main() {
    printf("Hello World");

    return 0;
}

```

## Probem 3
Write a program which will print `Hello` and `World` in two different lines.

**Solution:**
```c
#include<stdio.h>
int main() {
    printf("Hello\n");
    printf("World");

    return 0;
}

```
